# Autocomplete

**bash**:
```
_SIMNET_COMPLETE=source simnet > simnet_complete.sh
```

**zsh**:
```
_SIMNET_COMPLETE=source_zsh simnet > simnet_complete.sh
```

And then you put this into your .bashrc or .zshrc:
```
. /path/to/foo-bar-complete.sh
```
